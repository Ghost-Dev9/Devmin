Ressourcensteuerung
Eine Ressourcensteuerung ist eine Reihe von Beschränkungen für die Nutzung einer Ressource innerhalb der Zone und den zugehörigen Aktionen, wenn diese überschritten werden. Die verfügbaren Ressourcen zum Zeitpunkt des Schreibens sind:
zone.max-lwps
Begrenzt die Anzahl der Prozesse, die gleichzeitig innerhalb der Zone ausgeführt werden können.
zone.cpu-shares
Setzt eine Grenze für die Anzahl der Fair Share Scheduler (FSS) CPU-Anteile für eine Zone.
Für jede Ressource können Sie eine Reihe von Beschränkungen eingeben. Jede Beschränkung hat einen Privilegientyp (der immer Privilegiert ist), eine numerische Grenze und eine Aktion, die ausgeführt wird, wenn die Grenze überschritten wird. Die typische Aktion ist Verweigern, um jede Anfrage zu verhindern, die die Grenze überschreiten würde.
