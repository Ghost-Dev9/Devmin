Bacula-Gruppen-Backup-Jobs
Diese Seite listet Backup-Jobs auf, die für die gleichzeitige Ausführung auf allen Hosts innerhalb einer Bacula-Gruppe eingerichtet sind. Sie haben dieselben Einstellungen wie Standard-Backup-Jobs, mit der Ausnahme, dass keine Standardjobs zur gemeinsamen Nutzung von Einstellungen definiert werden können. Anstelle der Auswahl eines Clients für das Backup muss eine zuvor definierte Bacula-Gruppe ausgewählt werden.
