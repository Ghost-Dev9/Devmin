Bacula-Gruppen
Diese Seite listet Host-Gruppen auf, die jeweils eine Sammlung von Hosts enthalten, die gleichzeitig von einem Bacula-Gruppen-Backup-Job gesichert werden können. Die Hosts in einer Gruppe stammen entweder aus einer OCM-Manager-Knotengruppen-Datenbank oder aus dem Webmin-Servers-Index-Modul, abhängig von der Konfiguration dieses Moduls.

Damit Bacula-Gruppen-Backups funktionieren, muss auf jedem Host mindestens der Bacula-File-Daemon installiert sein, und alle Hosts in der Gruppe müssen dasselbe Passwort für diesen Director haben.
