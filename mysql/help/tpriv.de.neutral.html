Die folgenden Parameter sind verfügbar:
Tabelle
Die Tabelle, auf die diese Berechtigungen angewendet werden sollen.

Benutzername
Der/Die Benutzer:in, dem/der die Berechtigungen gewährt werden. Diese:r Benutzer:in muss bereits in der Benutzerberechtigungen-Tabelle existieren.

Hosts
Die Hosts, von denen aus der/die Benutzer:in auf die Tabelle zugreifen kann.

Tabellenberechtigungen
Die dem/der Benutzer:in gewährten Berechtigungen für alle Spalten der Tabelle.

Feldberechtigungen
Die maximalen Berechtigungen, die diesem/r Benutzer:in für eine beliebige Spalte der Tabelle auf der Seite Feldberechtigungen gewährt werden können.
