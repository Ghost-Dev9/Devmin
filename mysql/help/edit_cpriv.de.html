Feldberechtigungen bearbeiten
Diese Seite ermöglicht das Bearbeiten eines bestehenden Satzes von Spalten-Benutzer-Berechtigungen.Nach der Bearbeitung klicken Sie auf die Schaltfläche Speichern, um die Änderungen sofort zu übernehmen. Alternativ kann die Schaltfläche Löschen verwendet werden, um die Berechtigungen aus MySQL zu entfernen.
