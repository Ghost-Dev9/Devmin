Die folgenden Parameter sind verfügbar:
Tabelle
Die Tabelle, auf die diese Berechtigungen angewendet werden sollen.

Benutzername
der Benutzer, dem die Berechtigung gewährt werden. Dieser Benutzer muss bereits in der Benutzerberechtigungen-Tabelle existieren.

Hosts
Die Hosts, von denen aus der Benutzer auf die Tabelle zugreifen kann.

Tabellenberechtigungen
Die dem Benutzer gewährten Berechtigungen für alle Spalten der Tabelle.

Feldberechtigungen
Die maximalen Berechtigungen, die diesem/r Benutzer für eine beliebige Spalte der Tabelle auf der Seite Feldberechtigungen gewährt werden können.
