Die folgenden Parameter sind verfügbar:
Feld
Das Datenbankfeld, auf das diese Berechtigungen angewendet werden sollen.

Benutzername
der Benutzer, dem die Berechtigung gewährt werden. Dieser Benutzer muss bereits in der Benutzerberechtigungen-Tabelle existieren.

Hosts
Die Hosts, von denen aus der Benutzer auf die Tabelle zugreifen kann.

Berechtigungen
Die dem Benutzer gewährten Berechtigungen für die gewählte Spalte der Tabelle.
