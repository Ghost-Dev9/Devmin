Datenbankberechtigungen
Diese Seite ermöglicht die Festlegung, welche MySQL-Benutzer Zugriff auf welche Datenbanken von welchen Hosts aus haben. Jeder Benutzer muss bereits in der Benutzerberechtigungen-Tabelle existieren, kann jedoch hier zusätzliche Berechtigungen erhalten.

Benutzer können sich nur von den in Benutzerberechtigungen angegebenen Hosts aus anmelden. Es ist jedoch möglich, unterschiedliche Berechtigungen für verschiedene Hosts zu vergeben, indem mehrere Einträge für denselben/dieselbe Benutzer erstellt werden.
