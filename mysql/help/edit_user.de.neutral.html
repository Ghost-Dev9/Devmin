Benutzer:in bearbeiten
Dieses Formular ermöglicht das Bearbeiten eines bestehenden MySQL-Benutzer:innen-Kontos.Nach der Bearbeitung klicken Sie auf die Schaltfläche Speichern, um die Änderungen sofort zu übernehmen. Alternativ kann die Schaltfläche Löschen verwendet werden, um das Benutzer:innen-Konto aus MySQL zu entfernen.
