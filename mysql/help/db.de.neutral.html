Die folgenden Parameter sind verfügbar:
Datenbanken
Die Datenbanken, auf die der Zugriff erlaubt ist. Falls gewünscht, kann ein SQL-Regulärer Ausdruck verwendet werden.

Benutzername
Der/Die Benutzer:in, dem/der der Zugriff auf die oben genannte Datenbank gewährt wird. Diese:r Benutzer:in muss bereits in der Benutzerberechtigungen-Tabelle existieren.

Hosts
Die Hosts, von denen aus der/die Benutzer:in auf die Datenbank zugreifen kann. Falls die Option Von Host-Berechtigungen ausgewählt wird, sollte MySQL die Host-Berechtigungen zur Zugriffskontrolle verwenden.

Berechtigungen
Die MySQL-Operationen, die der/die Benutzer:in auf der Datenbank ausführen darf.
