Die folgenden Parameter sind verfügbar:
Datenbanken
Die Datenbanken, auf die der Zugriff erlaubt ist. Falls gewünscht, kann ein SQL-Regulärer Ausdruck verwendet werden.

Benutzername
der Benutzer, dem der Zugriff auf die oben genannte Datenbank gewährt wird. Dieser Benutzer muss bereits in der Benutzerberechtigungen-Tabelle existieren.

Hosts
Die Hosts, von denen aus der Benutzer auf die Datenbank zugreifen kann. Falls die Option Von Host-Berechtigungen ausgewählt wird, sollte MySQL die Host-Berechtigungen zur Zugriffskontrolle verwenden.

Berechtigungen
Die MySQL-Operationen, die der Benutzer auf der Datenbank ausführen darf.
