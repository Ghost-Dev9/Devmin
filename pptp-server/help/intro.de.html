PPTP VPN-Server
Dieses Modul ermöglicht es Ihnen, den PoPToP-Server so zu konfigurieren, dass Ihr System PPTP-Verbindungen von Clients akzeptiert. Auf der Hauptseite befinden sich Symbole für die Einstellung allgemeiner PPTP-Serveroptionen (wie die IP-Adressen, die den Clients zugewiesen werden), für die Festlegung von PPP-Optionen (wie die verwendete Authentifizierungsmethode), zum Bearbeiten von PPP-Konten, die zur Authentifizierung der Clients verwendet werden, und zum Anzeigen aktiver PPTP-Sitzungen.

Auf der Hauptseite befinden sich auch Schaltflächen zum Stoppen oder Starten des PPTP-Servers. Wenn der Server läuft, wird auch eine Schaltfläche Konfiguration anwenden angezeigt, um die aktuellen PPTP-Serveroptionen für neue Verbindungen zu aktivieren. Änderungen an den PPP-Optionen oder -Konten werden sofort auf alle neuen Verbindungen angewendet, die ab diesem Zeitpunkt hergestellt werden, ohne dass diese Schaltfläche gedrückt werden muss.
