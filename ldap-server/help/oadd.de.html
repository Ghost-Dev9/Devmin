Objekt erstellen
Diese Seite dient zum Erstellen eines neuen Objekts unter einem übergeordneten Objekt. Sie müssen mindestens den DN-Suffix für das Objekt eingeben, wie z.B. dc=people, sowie eine oder mehrere Objektklassen, wie z.B. top.

Wenn das Objekt anfängliche Attribute haben soll (die von den meisten Klassen verlangt werden), müssen diese in der Tabelle Andere Attribute eingegeben werden. Wenn erforderliche Attribute fehlen (oder nicht erlaubte Attribute angegeben wurden), wird beim Klicken auf die Schaltfläche Erstellen eine Fehlermeldung angezeigt.
