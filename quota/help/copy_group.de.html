Gruppenquoten kopieren
Diese Seite ermöglicht es, Quoten auf allen Dateisystemen für eine Gruppe auf eine oder mehrere andere Gruppen zu kopieren. Es gibt drei Möglichkeiten, die Zielgruppen auszuwählen:
Alle Gruppen
Quoten werden auf jede andere Gruppe auf Ihrem System kopiert.
Ausgewählte Gruppen
Quoten werden auf die ausgewählte(n) Gruppe(n) kopiert.
Gruppen mit Benutzern
Quoten werden auf alle Gruppen kopiert, in denen ein der ausgewählten Benutzer Mitglied ist.
