Benutzerquoten aktualisieren
Diese Seite kann verwendet werden, um die Block- oder Dateigrößenbegrenzungen für mehrere Benutzer gleichzeitig zu aktualisieren. Für jeden Quoten-Typ können Sie auswählen, ob er für den Benutzer unverändert bleiben soll, auf unbegrenzt gesetzt werden soll oder auf eine spezifische Grenze gesetzt werden soll.
