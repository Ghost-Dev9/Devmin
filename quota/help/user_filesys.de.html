Benutzer-Quotas
Diese Seite listet alle Dateisysteme auf, auf denen ein Benutzer eine Quota-Beschränkung hat. Ein Klick auf ein Dateisystem in der Liste ermöglicht die Bearbeitung des zugehörigen Quota-Werts, oder eine Auswahl aus dem Menü unterhalb der Liste kann getroffen und anschließend mit Quota bearbeiten bestätigt werden. Ist ein Dateisystem nicht in der Liste aufgeführt, bestehen keine Einschränkungen hinsichtlich der Anzahl der Dateien oder der belegbaren Speicherkapazität für diesen Benutzer auf dem Dateisystem.

Ein Klick auf die Schaltfläche Quotas kopieren ermöglicht das Kopieren aller Quota-Werte dieser:s Benutzer auf eine oder mehrere andere Benutzer. Dies ist nützlich, wenn mehreren Benutzer die gleiche Menge an Speicherplatz zugewiesen werden soll.
