Gruppenquoten
Diese Seite listet alle Dateisysteme auf, auf denen eine Gruppe eine Quote hat. Sie können auf ein Dateisystem aus der Liste klicken, um dessen Quote zu bearbeiten, oder ein Dateisystem aus dem Menü unten auswählen und auf Quote bearbeiten klicken. Wenn ein Dateisystem nicht in der Liste enthalten ist, gibt es keine Begrenzung für die Anzahl der Dateien oder den belegten Speicherplatz, den die Gruppe auf diesem Dateisystem belegen kann.

Durch Klicken auf die Schaltfläche Quoten kopieren können Sie alle Quoten für diese Gruppe auf eine oder mehrere andere Gruppen kopieren. Dies kann nützlich sein, wenn Sie mehreren Gruppen denselben Speicherplatz zuweisen möchten.
