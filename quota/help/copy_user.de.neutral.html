Benutzerquoten kopieren
Diese Seite ermöglicht es, Quoten auf allen Dateisystemen für einen Benutzer auf einen oder mehrere andere Benutzer zu kopieren. Es gibt drei Möglichkeiten, die Zielbenutzer auszuwählen:
Alle Benutzer
Quoten werden auf alle anderen Benutzer auf Ihrem System kopiert.
Ausgewählte Benutzer
Quoten werden auf den/die ausgewählte(n) Benutzer:innen kopiert.
Mitglieder ausgewählter Gruppen
Quoten werden auf alle Benutzer:innen kopiert, die eine der ausgewählten Gruppen als ihre primäre oder sekundäre Gruppe haben.
