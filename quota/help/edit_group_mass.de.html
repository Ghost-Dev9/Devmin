Gruppenquoten aktualisieren
Diese Seite kann verwendet werden, um die Block- oder Dateigrößenbegrenzungen für mehrere Gruppen gleichzeitig zu aktualisieren. Für jeden Quoten-Typ können Sie auswählen, ob er für die Gruppe unverändert bleiben soll, auf unbegrenzt gesetzt werden soll oder auf eine spezifische Grenze gesetzt werden soll.
