Benutzer-Überschreitungszeiten
Überschreitungszeiten bestimmen, wie lange ein Benutzer das Soft-Limit überschreiten darf, bevor die Nutzung zusätzlichen Speicherplatzes oder das Erstellen weiterer Dateien verhindert wird. Sobald ein Benutzer das Soft-Block-Limit überschreitet, hat er:sie eine festgelegte Zeitspanne, um die Speichernutzung unter das Limit zu reduzieren, bevor weitere Blöcke gesperrt werden. Dasselbe gilt für das Soft-Datei-Limit.
