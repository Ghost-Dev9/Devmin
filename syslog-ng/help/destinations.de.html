Log-Ziele
Jedes Log-Ziel ist eine Datei, ein Programm, eine Pipe, ein entfernter Server oder eine Benutzergruppe, an die Nachrichten gesendet werden können. Jedes Ziel hat einen eindeutigen Namen und genau eine Ausgabedatei, ein Programm oder ein anderes Ziel. Beim Erstellen oder Bearbeiten eines Ziels können verschiedene Optionen festgelegt werden, wie z. B. die Berechtigungen für die Ausgabedatei, die Portnummer eines entfernten Servers und weitere Einstellungen.

Diese Seite listet alle vorhandenen Log-Ziele auf und kann verwendet werden, um neue zu erstellen oder bestehende zu bearbeiten und zu löschen. Beim Entfernen oder Ändern von standardmäßig eingerichteten Zielen ist Vorsicht geboten, da diese oft kritische Log-Nachrichten speichern oder für den Betrieb des Systems wichtig sind. Das Hinzufügen eigener Log-Ziele ist jedoch unproblematisch.

Auf dieser Seite kann zudem der Inhalt von dateibasierten Log-Zielen angezeigt werden, indem auf den „Anzeigen“-Link auf der rechten Seite der Liste geklickt wird.
