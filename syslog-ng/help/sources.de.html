Log-Quellen
Eine Log-Quelle definiert verschiedene Methoden, über die Syslog-NG Log-Nachrichten empfangen kann, z. B. von anderen Systemen im Netzwerk, von lokalen Programmen über eine Unix-Socket oder vom Linux-Kernel. Jede Quelle hat einen eindeutigen Namen und kann eine oder mehrere Datenquellen verwenden.

Diese Seite listet alle bekannten Quellen auf und ermöglicht das Hinzufügen neuer, die Bearbeitung bestehender oder das gleichzeitige Löschen mehrerer Quellen. Beim Erstellen oder Bearbeiten einer Quelle muss angegeben werden, welche Datentypen sie nutzt, und die entsprechenden Parameter müssen korrekt ausgefüllt werden. In den meisten Fällen ist die voreingestellte Standardquelle bereits optimal konfiguriert und erfordert nur selten Änderungen – mit Ausnahme der Aktivierung des Empfangs von Nachrichten über das Netzwerk.
