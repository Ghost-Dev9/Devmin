Profile
Ein Ausführungsprofil ist ein Mechanismus, der verwendet wird, um die Befehle und Autorisierungen zusammenzufassen, die erforderlich sind, um eine bestimmte Funktion auszuführen. Jedes der auf dieser Seite aufgelisteten Profile hat eine Reihe von Autorisierungen und eine Liste von Unterprofilen, deren Autorisierungen es ebenfalls erbt.

Sie möchten möglicherweise zusätzliche Autorisierungen zu Ihrem System hinzufügen, wenn keine der hier aufgelisteten die genaue Zugriffskontrolle bietet, die Sie benötigen. Bestehende Standardautorisierungen sollten jedoch mit Vorsicht geändert werden, da ihre Änderung die Privilegien von Systembenutzern wie root und lp beeinträchtigen kann.
