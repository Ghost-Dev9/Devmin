Benutzereigenschaften
Diese Seite kann verwendet werden, um Profile und Autorisierungen Unix-Benutzern zuzuweisen, um ihnen zusätzliche Privilegien zu gewähren. Ein Profil definiert eine Reihe von Autorisierungen, die dem Benutzer gewährt werden, und Befehle, die der Benutzer mit höheren Privilegien ausführen kann (genannt Ausführungsprofile).

Zusätzlich kann dieser Abschnitt verwendet werden, um Rollen zu erstellen, die ebenfalls Profile und Autorisierungen haben können. Sobald eine Rolle erstellt ist, kann ein Benutzer alle Privilegien der Rolle erben, was es einfach macht, denselben Zugriff auf mehrere Benutzer zu gewähren.
