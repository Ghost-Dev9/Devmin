Ausführungsprofile
Ein Ausführungsprofil definiert einen Befehl, der mit einem vorhandenen Profil verknüpft ist und mit anderen Unix-Benutzer- und Grupperechten ausgeführt wird als der Benutzer, der ihn tatsächlich ausführt. Sie bieten eine ähnliche Funktionalität wie setuid-Programme, jedoch mit unterschiedlichem Verhalten je nach Benutzer.

Jedes Ausführungsprofil hat ein zugehöriges Profil, einen Befehl, auf den es angewendet wird, und eine Reihe von Benutzer- und Grupperechten, mit denen der Befehl ausgeführt wird. Diese zusätzlichen Rechte werden nur verwendet, wenn der Befehl aus einer profilbewussten Shell wie pfcsh oder pfsh ausgeführt wird.

Wenn Sie eigene Profile erstellt haben, können Sie diese Seite verwenden, um Befehle hinzuzufügen, die auf ihnen ausgeführt werden. Ausführungsprofile, die standardmäßig mit dem Betriebssystem geliefert werden, sollten jedoch nicht geändert werden, es sei denn, Sie wissen genau, was Sie tun.
