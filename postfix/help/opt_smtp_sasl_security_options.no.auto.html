SASL-sikkerhetsalternativer "."
smtp_sasl_security_options
Denne parameteren angir sikkerhetsalternativene som Postfix SMTP-klienten bruker for SASL-autentisering når den kobler til en ekstern SMTP-server.

Listen er en kommaseparert parameter for ett eller flere av følgende elementer:

  • noplaintext : ikke tillat metoder som bruker passord i klartekst
  • noactive : ikke tillate metoder som er utsatt for aktive (ikke-ordbok) angrep
  • nodictionary : forbyr metoder som er utsatt for passive (ordbok) angrep
  • noanonymous : ikke tillat metoder som tillater anonym autentisering
  • mutual_auth : tillat kun metoder som gir gjensidig autentisering (ikke tilgjengelig med SASL versjon 1)
