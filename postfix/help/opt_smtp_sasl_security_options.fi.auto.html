SASL-suojausasetukset "."
smtp_sasl_security_options
Tämä parametri määrittää suojausasetukset, joita Postfix SMTP -asiakasohjelma käyttää SASL-todennukseen muodostaessaan yhteyden etä-SMTP-palvelimeen.

Lista on pilkulla erotettu parametri, joka sisältää yhden tai useamman seuraavista kohteista:

  • noplaintext : estää metodit, jotka käyttävät selkokielisiä salasanoja
  • noactive : estää aktiiviselle (ei-sanakirja) hyökkäykselle alttiita metodeja
  • nodictionary : estää passiivisen (sanakirja)hyökkäyksen kohteena olevat metodit
  • noanonymous : estä menetelmät, jotka sallivat anonyymin todennuksen
  • mutual_auth : salli vain metodit, jotka tarjoavat molemminpuolisen todennuksen (ei käytettävissä SASL-versiossa 1)
