Параметри безпеки SASL.
smtp_sasl_security_options
Цей параметр визначає параметри безпеки, які клієнт Postfix SMTP використовує для SASL-автентифікації під час підключення до віддаленого SMTP-сервера.

Список – це параметр, розділений комами, одного або кількох із наступних елементів:

  • noplaintext : заборонити методи, що використовують паролі у відкритому тексті
  • noactive : заборонити методи, що підлягають активній (несловниковій) атаці
  • nodictionary : заборонити методи, що підлягають пасивній (словниковій) атаці
  • noanonymous : заборонити методи, що дозволяють анонімну автентифікацію
  • mutual_auth : дозволяє лише методи, що забезпечують взаємну автентифікацію (недоступно з SASL версії 1)
