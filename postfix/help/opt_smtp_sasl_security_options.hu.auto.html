SASL biztonsági beállítások "."
smtp_sasl_security_options
Ez a paraméter határozza meg azokat a biztonsági beállításokat, amelyeket a Postfix SMTP kliens a SASL hitelesítéshez használ, amikor egy távoli SMTP szerverhez csatlakozik.

A lista egy vesszővel elválasztott paraméter, amely a következő elemek közül egyet vagy többet tartalmaz:

  • noplaintext : letiltja azokat a metódusokat, amelyek sima szöveges jelszavakat használnak
  • noactive : letiltja az aktív (nem szótári) támadásnak kitett metódusokat
  • nodictionary : letiltja a passzív (szótár) támadásnak kitett metódusokat
  • noanonymous : letiltja azokat a metódusokat, amelyek lehetővé teszik az anonim hitelesítést
  • mutual_auth : csak a kölcsönös hitelesítést biztosító metódusokat engedélyezi (nem érhető el a SASL 1-es verziójával)
