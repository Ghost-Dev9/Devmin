SASL 보안 옵션 "."
smtp_sasl_security_options
이 매개변수는 Postfix SMTP 클라이언트가 원격 SMTP 서버에 연결할 때 SASL 인증에 사용하는 보안 옵션을 지정합니다.

목록은 다음 항목 중 하나 이상을 쉼표로 구분한 매개변수입니다.

  • noplaintext : 일반 텍스트 비밀번호를 사용하는 메서드를 허용하지 않습니다.
  • noactive : 활성(비사전) 공격에 노출된 메서드를 허용하지 않음
  • nodictionary : 수동(사전) 공격에 노출된 메서드를 허용하지 않음
  • noanonymous : 익명 인증을 허용하는 방법을 허용하지 않습니다.
  • mutual_auth : 상호 인증을 제공하는 방법만 허용합니다(SASL 버전 1에서는 사용할 수 없음)
