Параметры безопасности SASL "."
smtp_sasl_security_options
Этот параметр определяет параметры безопасности, которые использует клиент Postfix SMTP для аутентификации SASL при подключении к удаленному SMTP-серверу.

Список представляет собой разделенный запятыми параметр, состоящий из одного или нескольких из следующих элементов:

  • noplaintext : запретить методы, использующие пароли в виде открытого текста
  • noactive : запретить методы, подверженные активным (не словарным) атакам
  • nodictionary : запретить методы, подверженные пассивной (словарной) атаке
  • noanonymous : запретить методы, которые разрешают анонимную аутентификацию
  • mutual_auth : разрешить только методы, обеспечивающие взаимную аутентификацию (недоступно в SASL версии 1)
