Opciones de seguridad SASL "."
smtp_sasl_security_options
Este parámetro especifica las opciones de seguridad que utiliza el cliente SMTP de Postfix para la autenticación SASL cuando se conecta a un servidor SMTP remoto.

La lista es un parámetro separado por comas de uno o más de los siguientes elementos:

  • noplaintext : no permitir métodos que utilicen contraseñas de texto sin formato
  • noactive : no permite métodos sujetos a ataques activos (no de diccionario)
  • nodictionary : no permitir métodos sujetos a ataques pasivos (diccionario)
  • noanonymous : no permitir métodos que permitan la autenticación anónima
  • mutual_auth : solo permite métodos que proporcionen autenticación mutua (no disponible con la versión 1 de SASL)
