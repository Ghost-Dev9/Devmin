Επιλογές ασφαλείας SASL "."
smtp_sasl_security_options
Αυτή η παράμετρος καθορίζει τις επιλογές ασφαλείας που χρησιμοποιεί το πρόγραμμα-πελάτης SMTP της Postfix για τον έλεγχο ταυτότητας SASL κατά τη σύνδεση σε έναν απομακρυσμένο διακομιστή SMTP.

Η λίστα είναι μια παράμετρος διαχωρισμένη με κόμμα ενός ή περισσότερων από τα ακόλουθα στοιχεία:

  • noplaintext : απαγορεύονται οι μέθοδοι που χρησιμοποιούν κωδικούς πρόσβασης απλού κειμένου
  • noactive : απαγορεύει τις μεθόδους που υπόκεινται σε ενεργή (μη λεξικό) επίθεση
  • nodictionary : απαγορεύει τις μεθόδους που υπόκεινται σε παθητική (λεξικό) επίθεση
  • noanonymous : απαγορεύονται οι μέθοδοι που επιτρέπουν την ανώνυμη πιστοποίηση
  • mutual_auth : επιτρέπει μόνο μεθόδους που παρέχουν αμοιβαίο έλεγχο ταυτότητας (δεν είναι διαθέσιμο με την έκδοση 1 του SASL)
