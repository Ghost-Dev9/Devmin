خيارات أمان SASL "."
smtp_sasl_security_options
تحدد هذه المعلمة خيارات الأمان التي يستخدمها عميل Postfix SMTP لمصادقة SASL عند الاتصال بخادم SMTP بعيد.

القائمة عبارة عن معلمة مفصولة بفاصلة لعنصر واحد أو أكثر من العناصر التالية:

  • noplaintext : منع الأساليب التي تستخدم كلمات مرور نصية عادية
  • noactive : عدم السماح بالطرق المعرضة للهجوم النشط (غير القاموس)
  • nodictionary : منع الأساليب المعرضة للهجوم السلبي (القاموس)
  • noanonymous : عدم السماح بالطرق التي تسمح بالمصادقة المجهولة
  • mutual_auth : السماح فقط بالطرق التي توفر مصادقة متبادلة (غير متوفر مع إصدار SASL 1)
