SASL セキュリティ オプション "."
smtp_sasl_security_options
このパラメータは、Postfix SMTP クライアントがリモート SMTP サーバーに接続するときに SASL 認証に使用するセキュリティ オプションを指定します。

リストは、次の 1 つ以上の項目をコンマで区切って指定したパラメータです。

  • noplaintext : プレーンテキストのパスワードを使用する方法を禁止する
  • noactive : アクティブ（非辞書）攻撃を受けるメソッドを禁止する
  • nodictionary : パッシブ（辞書）攻撃を受けるメソッドを禁止する
  • noanonymous : 匿名認証を許可するメソッドを禁止する
  • mutual_auth : 相互認証を提供する方法のみを許可する (SASL バージョン 1 では使用できません)
