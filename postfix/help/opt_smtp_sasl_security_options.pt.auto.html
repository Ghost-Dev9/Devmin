Opções de segurança SASL "."
smtp_sasl_security_options
Este parâmetro especifica as opções de segurança que o cliente SMTP do Postfix usa para autenticação SASL ao se conectar a um servidor SMTP remoto.

A lista é um parâmetro separado por vírgulas de um ou mais dos seguintes itens:

  • noplaintext : não permite métodos que usam senhas em texto simples
  • noactive : não permite métodos sujeitos a ataque ativo (não dicionário)
  • nodictionary : não permite métodos sujeitos a ataque passivo (dicionário)
  • noanonymous : não permite métodos que permitem autenticação anônima
  • mutual_auth : permitir apenas métodos que forneçam autenticação mútua (não disponível com SASL versão 1)
