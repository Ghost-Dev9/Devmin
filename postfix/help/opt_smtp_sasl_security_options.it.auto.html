Opzioni di sicurezza SASL "."
smtp_sasl_security_options
Questo parametro specifica le opzioni di sicurezza che il client SMTP Postfix utilizza per l'autenticazione SASL quando si connette a un server SMTP remoto.

L'elenco è un parametro separato da virgole di uno o più dei seguenti elementi:

  • noplaintext : non consentire metodi che utilizzano password in chiaro
  • noactive : non consentire metodi soggetti ad attacchi attivi (non dizionario)
  • nodictionary : non consentire metodi soggetti ad attacchi passivi (dizionario)
  • noanonymous : non consentire metodi che consentono l'autenticazione anonima
  • mutual_auth : consente solo metodi che forniscono l'autenticazione reciproca (non disponibile con SASL versione 1)
