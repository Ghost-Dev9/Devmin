Opcions de seguretat SASL "."
smtp_sasl_security_options
Aquest paràmetre especifica les opcions de seguretat que el client SMTP de Postfix utilitza per a l'autenticació SASL quan es connecta a un servidor SMTP remot.

La llista és un paràmetre separat per comes d'un o més dels elements següents:

  • noplaintext : no permet mètodes que utilitzen contrasenyes de text pla
  • noactive : desactiva mètodes subjectes a atacs actius (no de diccionari)
  • nodictionary : desactiva mètodes subjectes a atac passiu (diccionari)
  • noanonymous : no permet mètodes que permeten l'autenticació anònima
  • mutual_auth : només permet mètodes que proporcionen autenticació mútua (no disponible amb SASL versió 1)
