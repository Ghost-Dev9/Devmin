Možnosti zabezpečenia SASL.
smtp_sasl_security_options
Tento parameter určuje možnosti zabezpečenia, ktoré klient Postfix SMTP používa na overenie SASL pri pripájaní k vzdialenému serveru SMTP.

Zoznam je parameter jednej alebo viacerých z nasledujúcich položiek oddelený čiarkami:

  • noplaintext : zakázať metódy, ktoré používajú heslá v otvorenom texte
  • noactive : zakázať metódy vystavené aktívnemu (neslovníkovému) útoku
  • nodictionary : zakázať metódy vystavené pasívnemu (slovníkovému) útoku
  • noanonymous : zakázať metódy, ktoré povoľujú anonymné overovanie
  • mutual_auth : povoliť iba metódy, ktoré poskytujú vzájomné overovanie (nie je dostupné so SASL verziou 1)
