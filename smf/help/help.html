SMF Modul

Diese Hilfeseite ist unvollständig

Einführung

Dieses Modul ermöglicht es Ihnen, SMF-Dienste in Solaris 10 anzusehen, zu ändern, zu aktualisieren und hinzuzufügen. Die Hauptseite besteht aus einer Liste von Diensten, und Sie können eine bestimmte Kategorie auswählen, deaktivierte Dienste anzeigen, Einträge sortieren usw. Von hier aus können Sie auch ein Assistenten-ähnliches Dienstprogramm starten, das die Dienst-Erstellung verwaltet, den Dienststatus, Abhängigkeiten/abhängige Dienste anzeigen und die Eigenschaftsgruppen/-eigenschaften für jeden Dienst anzeigen.

Hauptfenster

Das Hauptfenster des SMF-Moduls listet standardmäßig die aktivierten Dienste auf dem System, zeigt deren Status, die FMRI, die den Dienst identifiziert, und die Beschreibung, die mit dem Dienst geliefert wird. Sie können Statusänderungen auf mehrere Dienste anwenden, indem Sie die gewünschten Dienste auswählen und auf Aktivieren/Deaktivieren usw. klicken. Sie können auch die angegebenen Dienstinstanzen löschen, aber vermeiden Sie dies natürlich für wichtige Dienste.

Instanz-Viewer-Fenster

Ein Klick auf die FMRI einer bestimmten Instanz im Hauptfenster öffnet den Instanz-Viewer, der den Instanznamen/-beschreibung, den Status, alle zugehörigen Handbuchseiten (die angeklickt und angezeigt werden können), Erklärungen zu ungewöhnlichen Status (die angeklickt und angezeigt werden können) und Verweise auf zugehörige Logdateien (die ebenfalls angeklickt und angezeigt werden können) anzeigt. Von hier aus ist es auch möglich, die Abhängigkeits-/Abhängigkeitsbäume für die Instanz anzuzeigen, indem Sie den entsprechenden Link anklicken, und Eigenschaftsgruppen/-eigenschaften anzuzeigen/bearbeiten, indem Sie ebenfalls den entsprechenden Link anklicken.

Eigenschaftsgruppen/-Eigenschafts-Editor-Fenster

Ein Klick auf den Link "Eigenschaftsgruppen/-Eigenschaften anzeigen/bearbeiten" im Instanz-Viewer öffnet den Eigenschaftsgruppen-Editor, der die mit der ausgewählten FMRI verbundenen Eigenschaftsgruppen anzeigt. Sie können Eigenschaftsgruppen hier hinzufügen/löschen/ändern und dasselbe für die zugehörigen Eigenschaften tun, indem Sie auf die relevante Eigenschaftsgruppe im Eigenschaftsgruppen-Editor klicken.

Abhängigkeits-/Abhängigkeits-Viewer-Fenster

Ein Klick auf den Link "Abhängigkeits-/Abhängigkeitsbaum anzeigen" im Instanz-Viewer öffnet den Abhängigkeits-/Abhängigkeitsbaum für diese Instanz. Standardmäßig zeigt die Ansicht Abhängigkeiten/Abhängige der ersten Ebene, aber durch Klicken auf die + Schaltfläche können Sie den Baum erweitern, um zu sehen, was von den Diensten abhängt, die vom Dienst abhängen usw. Ähnlich wie im Hauptfenster können Statusänderungen auf mehrere Dienste angewendet werden, indem Kontrollkästchen verwendet werden, um die Dienste auszuwählen, die Sie ändern möchten. Ein Klick auf eine Instanz-FMRI bringt den Abhängigkeits-/Abhängigkeitsbaum für diese Instanz an.

Dienst-Erstellungs-Assistent

Ein Klick auf die Schaltfläche "Neuen Dienst erstellen" öffnet den SMF-Dienst-Erstellungsassistenten, der die erforderlichen Informationen sammelt, um einen neuen SMF-Dienst zu erstellen, es in ein Manifest formatiert und es in das SMF-Repository importiert. Ein erfolgreicher Import führt dazu, dass der Dienst in der Hauptdienstliste erscheint.

Service-Instanz-Suchformular

Ein rudimentäres FMRI-Suchwerkzeug wird bereitgestellt, bei dem es ausreicht, einen Teil des Dienst-FMRI einzugeben, um alle passenden FMRIs zu finden, z.B. "nfs" ruft alle Dienste mit "nfs" im FMRI ab. Eine "Durchsuchen"-Funktion ist ebenfalls verfügbar.
