Diese Hilfeseite ist unvollständig

Abhängigkeits-/Abhängigkeitsbaum-Viewer

Die auf dieser Seite angezeigten Informationen entsprechen den Abhängigkeits-/Abhängigkeitsbaumdaten, die mit svcs -d/-D gesammelt wurden, d.h. welche Dienste von diesem Dienst abhängen und von welchen Diensten dieser abhängt. Statusänderungen können auf Abhängige/Abhängigkeiten angewendet werden, und das Klicken auf eine Abhängige/Abhängigkeit-FMRI bringt Sie zur Seite des Statuseditors.

Die Abhängigkeits-/Abhängigkeitsbäume können auf jede gewünschte Ebene durch Klicken auf die "+"- und "-" Schaltflächen erweitert oder eingeklappt werden. Statusänderungen können auf den Dienst und ausgewählte Abhängigkeiten/Abhängige mit Kontrollkästchen angewendet werden, um die Dienste auszuwählen.

Ein Klick auf eine FMRI im Abhängigkeits-/Abhängigkeitsbaum führt Sie zum Abhängigkeits-/Abhängigkeitsbaum für diese Instanz.
