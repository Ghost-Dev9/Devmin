Alias-Ziel
Jeder Alias kann E-Mails an mehrere Empfänger weiterleiten, wobei jeder Empfänger einer der folgenden Typen sein kann:
E-Mail-Adresse
Entweder eine vollständige E-Mail-Adresse wie foo@bar.com oder der Name eines anderen Benutzers auf dem System, z. B. jcameron.
Adressen in Datei
Für diesen Alias-Typ muss der vollständige Pfad zu einer Datei angegeben werden, die eine Liste von E-Mail-Adressen enthält (eine Adresse pro Zeile). Jede Adresse kann entweder vollständig qualifiziert sein oder den Namen eines anderen Systembenutzers enthalten.
In Datei schreiben
In diesem Fall wird die E-Mail in die angegebene Datei geschrieben. Um alle E-Mails an eine Adresse zu verwerfen, kann /dev/null als Dateiname angegeben werden.
An Programm weiterleiten
Jedes Mal, wenn eine E-Mail an diesen Alias gesendet wird, wird das angegebene Programm ausgeführt, und die gesamte E-Mail (Header und Body) wird über die Standardeingabe an das Programm übergeben.
