Benutzer-Weiterleitungsdateien
Der für diese Option angegebene Wert muss eine durch Doppelpunkte getrennte Liste von Pfaden sein, die bestimmt, wo Sendmail die individuelle Weiterleitungsdatei von Benutzer beim Zustellen von E-Mails sucht. Die Variablen $z für das Home-Verzeichnis der Benutzer und $w für den System-Hostnamen können im Pfad verwendet werden.
