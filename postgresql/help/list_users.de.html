PostgreSQL Benutzer
Auf dieser Seite können Sie Benutzer erstellen und bearbeiten, die Zugriff auf Ihre PostgreSQL-Datenbanken haben. Für jeden Benutzer können Sie einen Benutzernamen, ein optionales Passwort, ein optionales Ablaufdatum festlegen und auswählen, ob der Benutzer andere Benutzer oder Datenbanken erstellen darf.
