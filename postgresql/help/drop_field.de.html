Feld löschen
Auf dieser Seite können Sie ein bestehendes Tabellenfeld löschen. Klicken Sie auf die Radiotaste nach dem Feld, um anzugeben, welches Feld gelöscht werden soll. Bestätigen Sie auch im Kontrollkästchen, dass Sie die Löschung durchführen möchten und dass alle Daten in dieser Spalte verloren gehen. Beachten Sie, dass alle Definitionen (wie eine Definition des Standardwerts) in der Tabelle verloren gehen, außer den Spaltennamen, -typen und -größen. Klicken Sie auf die Schaltfläche, um das Feld zu löschen.
