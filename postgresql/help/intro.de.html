PostgreSQL-Datenbankserver
PostgreSQL ist ein einfacher Datenbankserver für Unix-Systeme. Er unterstützt mehrere Datenbanken, leistungsstarke Zugriffskontrollen, viele Datentypen und die meisten Standard-SQL-Syntax. Jede vom Server verwaltete Datenbank kann mehrere Tabellen enthalten, und jede Tabelle enthält mehrere Datenzeilen.

Die Hauptseite zeigt oben eine Liste von Datenbanksymbolen, mit einem Link zum Hinzufügen einer neuen Datenbank darunter. Jeder PostgreSQL-Server hat mindestens eine Vorlagen-Datenbank, die üblicherweise template1 genannt wird. Unter den Datenbanken befinden sich Symbole zur Konfiguration von Zugriffskontrollen und Berechtigungen, und unten befindet sich eine Schaltfläche zum Starten oder Stoppen des Datenbankservers.
