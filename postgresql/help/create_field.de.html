Feld hinzufügen
Auf dieser Seite können Sie ein neues Feld zur Tabelle hinzufügen. Die bearbeitbaren Parameter für das neue Feld sind :
Feldname
Der Name dieses Feldes in der Tabelle.

Typbreite (für varchar und char Felder)
Die maximale Anzahl an Zeichen, die für Daten in diesem Feld erlaubt sind.

Array-Feld?
Wenn diese Option auf Ja gesetzt ist, kann das Feld mehrere Werte des gleichen Typs speichern.

Wenn Sie fertig sind, klicken Sie auf die Speichern-Schaltfläche, um die Tabellenstruktur mit dem neuen Feld zu aktualisieren.
