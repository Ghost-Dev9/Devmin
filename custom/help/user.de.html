Als Benutzer ausführen
Der Unix-Benutzer, unter dem der Befehl ausgeführt wird. Es sollte stets mit dem niedrigstmöglichen Berechtigungslevel gearbeitet werden – führen Sie Befehle nicht als root aus, es sei denn, es ist wirklich erforderlich, insbesondere wenn der Befehl von anderen Webmin-Benutzer ausgeführt werden soll.
