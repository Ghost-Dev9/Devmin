Einzuhängendes Verzeichnis
Dies ist das vom NFS-Server exportierte Verzeichnis, das auf dem eigenen Host eingehängt wird.
Die Schaltfläche zeigt eine Liste der vom Server exportierten Verzeichnisse an.
Für NFSv4 sollte immer / eingehängt werden, da es sich um das vom Server exportierte Pseudo-Dateisystem handelt. Alle verfügbaren exportierten Verzeichnisse befinden sich in diesem Pseudo-Dateisystem.
