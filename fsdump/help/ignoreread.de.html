Lesefehler bei Dateien ignorieren?
Wenn diese Option auf Ja gesetzt ist, werden alle Fehler beim Lesen von Dateien, die in das Backup aufgenommen werden sollen, ignoriert. Dies ist nützlich, um den normalerweise unnützen Fehler file changed as we read it zu überspringen, kann jedoch Berechtigungsprobleme oder Dateikorruptionsprobleme verschleiern.
