Wiederherstellen von Datei oder Gerät
Diese Option bestimmt, wo die Backup-Dump-Daten gelesen werden. Es kann entweder eine lokale Datei oder ein Bandgerät oder eine Datei oder ein Gerät auf einem Remote-Host sein. Damit Wiederherstellungen von einem Remote-Host funktionieren, muss dieser das rmt-Protokoll unterstützen, das aus Sicherheitsgründen normalerweise nicht standardmäßig aktiviert ist.
