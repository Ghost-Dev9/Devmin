Bandgröße
Wenn 'Automatisch berechnen' gewählt wird, wird das Backup so lange auf das Band geschrieben, bis der Speicherplatz auf dem Band oder Dateisystem erschöpft ist. Alternativ können Sie die korrekte Anzahl an verfügbaren Kilobytes angeben, was nützlich sein kann, wenn die automatische Größenerkennung ein falsches Ergebnis liefert.
