Nur Backup mit Bezeichnung wiederherstellen
Beim Wiederherstellen von einem Band, das mehrere Backups enthält, kann diese Option verwendet werden, um genau anzugeben, welches Backup wiederhergestellt werden soll. Die Bezeichnung, die Sie eingeben, muss mit der Bezeichnung übereinstimmen, die beim ursprünglichen Erstellen des Backups festgelegt wurde.
