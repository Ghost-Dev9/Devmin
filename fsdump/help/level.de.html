Dump-Level
Das Dump-Level steuert, welche Dateien in das Backup aufgenommen werden. Wenn Level 0 gewählt wird, werden alle Dateien unter dem Verzeichnis, das gesichert werden soll, gespeichert. Andernfalls werden nur Dateien geschrieben, die neu sind oder seit dem letzten Dump eines niedrigeren Levels geändert wurden.

Beim Sichern eines Verzeichnisses, das kein Dateisystem-Mountpunkt ist, kann nur Level 0 ausgewählt werden.
