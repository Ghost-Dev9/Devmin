Backup-Ziel
Dies kann entweder der Pfad zu einer Datei sein, in die das Backup geschrieben wird, der Pfad zu einem Bandlaufwerk-Gerät (wie /dev/st0) oder eine Datei oder ein Gerät auf einem entfernten Host. Backups auf ein entferntes System werden über das SSH-Protokoll oder optional das ältere unsichere RSH-Protokoll durchgeführt.
