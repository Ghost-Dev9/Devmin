Ersetzte Dateien speichern?
Gibt an, ob Softwaredateien, die während der Installation ersetzt werden, gespeichert werden sollen.

JA weist das System an, vorherige Versionen der ersetzten Software zu speichern. Falls die Installation fehlschlägt, ermöglicht diese Option dem Bereinigungsprozess, die vorherige Version der Software wiederherzustellen. Falls die Option „SOFTWARE VERBINDLICH ÜBERNEHMEN?“ auf NEIN gesetzt wurde, muss „ERSATZDATEIEN SPEICHERN?“ auf JA gesetzt werden. Das Speichern der ersetzten Dateien benötigt zusätzlichen Speicherplatz und verlängert die Installationsdauer, da zusätzliche Verarbeitung erforderlich ist.

NEIN weist das System an, vorherige Versionen der Software während der Installation nicht zu speichern. Falls die Installation fehlschlägt, kann die vorherige Version der Software NICHT wiederhergestellt werden und wird als BESCHÄDIGT (BROKEN) markiert. In diesem Fall muss das Software-Update erneut installiert werden.
