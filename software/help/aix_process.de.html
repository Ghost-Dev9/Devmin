Mehrere Datenträger verarbeiten?
Diese Option ermöglicht die Verarbeitung von Multi-Volume-CDs. Die Standardeinstellung ist Ja.

Die Installation von Software von einer CD-ROM-Quelle kann Dateisätze erfordern, die auf anderen CD-Datenträgern gespeichert sind. Wenn diese Option auf Ja gesetzt ist, können Dateisätze von mehreren CD-Volumen während derselben Sitzung verarbeitet werden. Wird die Option auf Nein gesetzt, werden selbst Multi-Volume-CDs als eigenständige CDs betrachtet, und Dateisätze auf anderen Datenträgern sind nicht zugänglich.
