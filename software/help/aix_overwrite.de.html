GLEICHE oder NEUERE Versionen ÜBERSCHREIBEN?

JA wählen, um dieselbe oder eine ältere Version einer Software erneut zu installieren, auch wenn bereits eine neuere Version auf dem System vorhanden ist. Beachte, dass diese Option nicht für Updates gilt – ein Überschreiben ist nur für Basis-Dateisätze möglich. Die Standardeinstellung ist NEIN.

Hinweis: Diese Option muss nicht auf JA gesetzt werden, wenn eine neuere Version installiert wird.

Falls diese Option auf JA gesetzt wird, muss die Option „ERFORDERLICHE Software AUTOMATISCH installieren?“ auf NEIN gesetzt werden.
