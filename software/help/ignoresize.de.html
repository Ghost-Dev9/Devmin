Dateisystemgrößen überprüfen?
Standardmäßig überprüft RPM bei Auswahl von Ja den freien Speicherplatz auf allen eingehängten Dateisystemen während der Installation eines Pakets, um sicherzustellen, dass genügend Platz vorhanden ist. Falls jedoch einige Dateisysteme von nicht erreichbaren NFS-Servern eingehängt sind, kann es sinnvoll sein, Nein zu wählen, damit das Paket trotzdem installiert werden kann.
