Installation überprüfen und Dateigrößen kontrollieren?
Gibt an, ob das System einige Prüfungen an der installierten Software durchführen soll.

JA weist das System an, eine Prüfsumme zu berechnen, um zu bestätigen, dass die Softwaredateien die korrekte Länge haben, die richtige Anzahl an Ziffern und Zeichen enthalten und dass alle Abhängigkeiten für die auf dem System installierte Software überprüft werden.

NEIN weist das System an, die Installation nicht zu überprüfen. Der Verifizierungsprozess kann die Installationszeit erheblich verlängern. Falls eine schnellere Installation gewünscht ist, sollte NEIN gewählt werden.
