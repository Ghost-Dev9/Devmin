NUR VORSCHAU? (Installation wird NICHT durchgeführt)
Gibt an, ob die Installation der ausgewählten Softwareprodukte und Updates lediglich in einer Vorschau angezeigt werden soll. Eine Vorschau ermittelt die Voraussetzungen für eine erfolgreiche Installation.

JA weist das System an, Informationen zu Speicherplatzanforderungen, das Ergebnis der Vorabprüfung und eine Liste der zu installierenden Softwareprodukte und Updates anzuzeigen. Diese Informationen können auch eine Liste von Softwareprodukten und Updates enthalten, die aufgrund fehlender Abhängigkeiten nicht installiert werden können. Die Standardeinstellung für diese Option ist NEIN.

Falls die Option „DETAILLIERTE Ausgabe?“ ebenfalls auf JA gesetzt wird, wird die Liste der Softwareprodukte und Updates in der Installationsreihenfolge angezeigt.

NEIN weist das System an, die Installation durchzuführen.
