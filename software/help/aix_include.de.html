Entsprechende SPRACH-Dateisätze einbeziehen?
Weist das System an, alle Sprachdateisätze für die aktuell installierten oder zuvor installierten Dateisätze einzubeziehen. Die Sprache, die während der Installation des Basisbetriebssystems verwendet wurde, wird übernommen.

Es wird empfohlen, diese Option auf JA zu belassen, wenn die Hauptsprache nicht Englisch ist. Diese Option wird ignoriert, wenn die LANG-Variable auf dem System C ist (LANG=C).
